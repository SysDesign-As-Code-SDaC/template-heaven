# Template Heaven CLI - Interactive Tool Guide

## ğŸ¯ Overview

Template Heaven is an **interactive CLI tool** for discovering templates and scaffolding projects (similar to Yeoman). It provides:

1. **Interactive Template Discovery** - Search and browse templates
2. **Scaffolding Generator** - Generate projects from templates with customization
3. **Beautiful Terminal UI** - Rich formatting with colors and tables

---

## ğŸš€ Quick Start

### Installation

The CLI is already installed! You can use it via:

```bash
# Direct command (if installed)
templateheaven init

# Or via Python module
python3 -m templateheaven.cli.main init
```

### Interactive Wizard (Recommended)

```bash
templateheaven init
```

This launches an interactive wizard that guides you through:
1. **Stack Selection** - Choose technology stack (frontend, backend, fullstack, etc.)
2. **Template Selection** - Browse and select from available templates
3. **Project Configuration** - Set project name, author, license, package manager
4. **Project Creation** - Scaffold the project with Jinja2 templating

---

## ğŸ“‹ Available Commands

### 1. Initialize Project (Interactive)

```bash
# Launch interactive wizard
templateheaven init

# Non-interactive mode (specify all options)
templateheaven init --template react-vite --name my-app --no-wizard

# With options
templateheaven init \
  --template react-vite \
  --name my-app \
  --stack frontend \
  --author "John Doe" \
  --license MIT \
  --package-manager npm \
  --directory ./projects
```

### 2. List Templates

```bash
# List all templates
templateheaven list

# Filter by stack
templateheaven list --stack frontend

# Search templates
templateheaven list --search "react typescript"

# Filter by tags
templateheaven list --tags react,typescript

# Output as JSON
templateheaven list --format json

# Limit results
templateheaven list --limit 10

# Search source options
templateheaven list --source local    # Local templates only
templateheaven list --source github   # GitHub search (if configured)
templateheaven list --source auto     # Auto-detect (default)
```

### 3. Search Templates

```bash
# Interactive search
templateheaven search

# Search with query
templateheaven search --query "machine learning"

# With options
templateheaven search \
  --query "react typescript" \
  --limit 5 \
  --min-score 0.5 \
  --source github
```

### 4. Get Template Info

```bash
# Show detailed template information
templateheaven info react-vite

# Shows:
# - Name, stack, description
# - Version, author, license
# - Tags, features, dependencies
# - Upstream URL
```

### 5. Configuration Management

```bash
# List all configuration
templateheaven config --list-all

# Set configuration
templateheaven config --key default_author --value "John Doe"
templateheaven config --key default_license --value "MIT"

# Unset configuration
templateheaven config --unset github_token

# Reset to defaults
templateheaven config --reset
```

### 6. Statistics

```bash
# Show template and cache statistics
templateheaven stats
```
