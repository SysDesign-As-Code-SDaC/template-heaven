# Basic Bravetto App

A minimal Bravetto application template to get you started quickly.

## Features

- ✅ Minimal setup
- ✅ Hot reloading
- ✅ TypeScript support
- ✅ Basic routing
- ✅ Component structure
- ✅ Development server

## Quick Start

```bash
# Copy this template
cp -r stacks/bravetto/templates/basic-bravetto-app my-app

# Navigate to project
cd my-app

# Install dependencies
npm install

# Start development server
npm run dev
```

## Project Structure

```
basic-bravetto-app/
├── src/
│   ├── components/     # Reusable components
│   ├── pages/         # Page components
│   ├── utils/         # Utility functions
│   └── main.ts        # Application entry point
├── public/            # Static assets
├── package.json       # Dependencies and scripts
├── tsconfig.json      # TypeScript configuration
└── README.md          # This file
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests
- `npm run lint` - Run linter

## Configuration

Edit `src/config/app.ts` to customize your application settings.

## Next Steps

1. Add your components in `src/components/`
2. Create pages in `src/pages/`
3. Add utilities in `src/utils/`
4. Customize styling and configuration
